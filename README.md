# Student Management System

A simple console-based Student Management System built using Python.

## Features

- Add Student
- Remove Student
- Search Student
- Display All Students
- Update Student Details
- Exit Program

## Concepts Used

- Lists
- Dictionaries
- Functions
- Loops
- If-Else Conditions

## How to Run

1. Install Python
2. Open terminal
3. Navigate to project folder

```bash
python main.py
```
